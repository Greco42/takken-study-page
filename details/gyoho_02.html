宅建業法 - 免許の有効期間と更新

重要ポイント要約

  • 免許の有効期間は5年間。
  • 免許の更新は、有効期間満了の90日前から30日前までに行う必要がある。
  • 免許換え（大臣免許⇔知事免許など）を受けた場合、有効期間は新たに5年間となる。
  • 商号・名称、役員、専任の宅建士などに変更があった場合は30日以内に届出が必要（免許換えではない）。
  • 廃業等の届出も重要。死亡、破産、解散などのケースを覚える。

詳細解説

1. 免許の有効期間

宅建業の免許の有効期間は、免許を受けた日から5年間です（宅建業法第3条第2項）。この5年という数字は必ず覚えてください。

2. 免許の更新

有効期間満了後も引き続き宅建業を営む場合は、免許の更新が必要です。更新の申請は、有効期間が満了する日の90日前から30日前までの間に行わなければなりません（宅建業法第3条第3項）。この期間も頻出です。

3. 変更の届出

免許を受けた宅建業者は、以下の事項に変更があった場合、その日から30日以内に、免許権者に届出をしなければなりません（宅建業法第9条）。

  • 商号または名称
  • 役員の氏名
  • 事務所の名称・所在地
  • 専任の宅地建物取引士の氏名

これは「免許換え」とは違い、免許証の書き換えに過ぎず、有効期間はリセットされません。

4. 廃業等の届出

宅建業を廃止した場合や、業者が死亡・破産・解散した場合なども、30日以内に届出が必要です。誰が届出をするのかが問われます。

  • 個人業者が死亡 → その相続人
  • 法人が合併により消滅 → 消滅した法人を代表する役員であった者
  • 法人が破産 → その破産管財人

知識チェック

問1. 宅地建物取引業の免許の有効期間は、免許を受けた日から何年間か。

  • 1. 1年間
  • 2. 3年間
  • 3. 5年間
  • 4. 10年間

正解は3

宅建業の免許の有効期間は5年間です。これは基本的な知識として必ず覚えておきましょう。

問2. 免許の更新申請を行う期間として正しいものはどれか。

  • 1. 有効期間満了の60日前から30日前まで
  • 2. 有効期間満了の90日前から60日前まで
  • 3. 有効期間満了の90日前から30日前まで
  • 4. 有効期間満了の30日前から満了日まで

正解は3

免許の更新申請は、有効期間満了の90日前から30日前までの間に行う必要があります。この「90日」「30日」という数字は重要です。

問3. 宅地建物取引業者（甲県知事免許）が、その代表取締役の氏名を変更した場合、どのような手続きが必要か。

  • 1. 14日以内に甲県知事に変更の届出をする。
  • 2. 30日以内に甲県知事に変更の届出をする。
  • 3. 遅滞なく甲県知事に変更の届出をする。
  • 4. 変更の届出は不要である。

正解は2

役員の氏名に変更があった場合、宅建業者はその日から30日以内に免許権者に変更の届出をする必要があります。

問4. 甲県知事免許を持つ宅建業者A社が、乙県にも事務所を設置し、国土交通大臣へ免許換えの申請を行った。この場合、A社の免許の有効期間はどうなるか。

  • 1. 甲県知事免許の残りの期間を引き継ぐ。
  • 2. 甲県知事免許の期間はリセットされ、新たに3年間となる。
  • 3. 国土交通大臣の免許を受けた日から、新たに5年間となる。
  • 4. 事務所の数に関わらず、有効期間は変わらない。

正解は3

免許換えが行われた場合、従前の免許は効力を失い、新たな免許の有効期間がその免許を受けた日から5年間となります。

問5. 個人である宅地建物取引業者Aが死亡した場合、廃業の届出は誰がいつまでに行う必要があるか。

  • 1. Aの相続人が、死亡を知った日から14日以内
  • 2. Aの相続人が、死亡を知った日から30日以内
  • 3. Aの専任の宅地建物取引士が、死亡日から30日以内
  • 4. 届出は不要である。

正解は2

個人業者が死亡した場合、その相続人が、死亡の事実を知った日から30日以内に、その旨を免許権者に届け出なければなりません。
