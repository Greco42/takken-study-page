法令上の制限 - 農地法

重要ポイント要約

  • 農地法は、国内の農地や採草放牧地を守るための法律。
  • 3条許可：農地を農地のまま、所有権を移転したり貸したりする場合。許可権者は農業委員会。
  • 4条許可：農地の所有者が、自らその農地を宅地などに転用する場合。許可権者は都道府県知事等。
  • 5条許可：農地を宅地などに転用する目的で、所有権を移転したり貸したりする場合。許可権者は都道府県知事等。
  • 市街化区域内の農地を転用する場合（4条・5条）は、許可は不要で、あらかじめ農業委員会へ届け出ればよい。

詳細解説

1. 農地法の目的と対象

農地法は、食料の安定供給の基盤である優良な農地を確保し、その効率的な利用を図ることを目的としています。対象となるのは「農地」と「採草放牧地」です。

2. 権利移動と転用の規制（3条・4条・5条）

農地法では、目的別に3種類の許可制度を設けています。

① 農地法3条（権利移動）

内容：農地や採草放牧地を、耕作目的で売買したり、貸し借りしたりする場合。
ポイント：農地のまま、利用する人（権利者）が変わるケース。
許可権者：農業委員会

② 農地法4条（転用）

内容：農地の所有者自らが、その農地を農地以外のもの（宅地、駐車場、資材置場など）にする場合。
ポイント：利用する人（権利者）は変わらず、土地の目的だけが変わるケース。
許可権者：都道府県知事等

③ 農地法5条（転用目的権利移動）

内容：農地を農地以外のものにする目的で、売買や貸し借りをする場合。
ポイント：利用する人（権利者）と土地の目的が同時に変わるケース。宅建業者が最も関わる取引。
許可権者：都道府県知事等

【市街化区域内の特例】

計画的に市街化を進める市街化区域においては、農地転用（4条・5条）のハードルが低くなっています。都道府県知事等の「許可」は不要で、あらかじめ農業委員会に「届出」をするだけで足ります。ただし、3条（農地のまま権利移動）については、市街化区域内であっても農業委員会の「許可」が必要です。

知識チェック

問1. 農業を営むAさんが、所有する農地を、農業を始めたいBさんに売却する場合、農地法上どのような手続きが必要か。

  • 1. 3条の規定による農業委員会の許可
  • 2. 4条の規定による都道府県知事の許可
  • 3. 5条の規定による都道府県知事の許可
  • 4. 許可も届出も不要

正解は1

農地を農地のまま（耕作目的で）売買するのは、権利移動にあたり、農地法3条の許可が必要です。許可権者は農業委員会です。

問2. Aさんが、自己所有の農地を、自分の息子Bの家を建てるために宅地に転用する場合、どのような手続きが必要か。

  • 1. 3条の規定による農業委員会の許可
  • 2. 4条の規定による都道府県知事の許可
  • 3. 5条の規定による都道府県知事の許可
  • 4. 届出をすればよい。

正解は2

所有者が変わらず、農地を宅地などに転用するのは農地法4条の規制対象です。許可権者は都道府県知事等です。

問3. 市街化区域内にある自己所有の農地を、駐車場にするために転用する場合、必要な手続きはどれか。

  • 1. 都道府県知事の許可
  • 2. 農業委員会の許可
  • 3. 農業委員会への事前の届出
  • 4. 手続きは不要

正解は3

市街化区域内の農地転用（4条・5条）については特例があり、許可は不要で、あらかじめ農業委員会へ届け出るだけで足ります。

問4. 農地法4条または5条の許可を受けずに農地を無断で転用した場合、どうなるか。

  • 1. 100万円以下の罰金が科される。
  • 2. 契約は有効だが、行政指導を受ける。
  • 3. 工事の中止や原状回復を命じられることがあり、契約も無効となる。
  • 4. 追認の届出をすれば、有効な転用として扱われる。

正解は3

無許可での転用は、原状回復命令や罰則（3年以下の懲役または300万円以下の罰金）の対象となるだけでなく、その原因となった売買契約等も無効となります。

問5. 相続によって農地の所有権を取得した場合、農地法3条の許可は必要か。

  • 1. 必要である。
  • 2. 不要であるが、農業委員会への届出が必要である。
  • 3. 不要であり、届出も必要ない。
  • 4. 家庭裁判所の許可が必要である。

正解は2

相続は当事者の意思に基づかない包括的な権利移転であるため、農地法3条の許可は不要です。ただし、農地の状況を把握するため、相続を知った時から遅滞なく、農業委員会へその旨を届け出る必要があります。
