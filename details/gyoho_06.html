宅建業法 - クーリング・オフ

重要ポイント要約

  • 適用要件：売主が宅建業者で、買主が宅建業者でないこと。
  • 申込みや契約をした場所が、事務所等以外であること。
  • クーリング・オフができる期間は、業者から書面で告知を受けた日から起算して8日間。
  • クーリング・オフの意思表示は書面で行う必要があり、その効力は書面を発信した時に生じる。
  • クーリング・オフができない場合：物件の引渡しを受け、かつ、代金の全部を支払った時。

詳細解説

1. クーリング・オフ制度とは

冷静な判断ができない状況（例：モデルルームでの見学で気分が高揚している、喫茶店で長時間勧誘されたなど）で契約の申込みや締結をしてしまった買主を保護するため、一定期間内であれば無条件で契約を解除できる制度です。これは自ら売主となる宅建業者に課される「8種制限」の一つです。

2. 適用される場面

以下の3つの条件を全て満たす場合に、クーリング・オフが適用される可能性があります。

  1. 売主が宅建業者であること。
  2. 買主が宅建業者でないこと（プロは保護の対象外）。
  3. 申込みまたは契約締結の場所が「事務所等」以外の場所であること。

「事務所等」とは、本店・支店、専任の宅建士を置くべき案内所などを指します。喫茶店や買主の自宅などは「事務所等以外」の場所に該当します。

3. クーリング・オフが「できなくなる」場合

たとえ適用要件を満たしていても、以下のいずれかに該当するとクーリング・オフはできなくなります。

  • 業者からクーリング・オフについて書面で告知を受けた日から起算して8日を経過したとき。
  • 宅地または建物の引渡しを受け、かつ、その代金の全部を支払ったとき。（「かつ」がポイント）

知識チェック

問1. クーリング・オフに関する次の記述のうち、宅地建物取引業法の規定によれば、買主が契約を解除できるものはどれか。（買主は宅建業者ではないものとする）

  • 1. 宅地建物取引業者である売主の事務所で買受けの申込みをし、翌日、喫茶店で契約を締結した場合。
  • 2. 買主の申出により、買主の自宅で契約を締結し、その場で物件の引渡しと代金全額の支払いを行った場合。
  • 3. テント張りの案内所で買受けの申込みをし、10日後に宅地建物取引業者である売主からクーリング・オフについて何も告げられずに契約を締結した場合。
  • 4. 買主は、クーリング・オフによる契約の解除を、口頭で売主の宅地建物取引業者に告げた場合。

正解は3

1は申込みが事務所なので不可。2は引渡しと代金全額支払で不可。4は書面で行う必要があるため不可。3はテント張りの案内所は「事務所等以外」の場所であり、書面での告知がない限り8日の期間は進行しないため解除できます。

問2. クーリング・オフの適用がないのは、次のうちどのケースか。

  • 1. 売主が宅建業者で、買主が個人。
  • 2. 売主が宅建業者で、買主も宅建業者。
  • 3. 売主が個人で、買主が宅建業者。
  • 4. 媒介業者が宅建業者で、売主・買主ともに個人。

正解は2

クーリング・オフ制度は、プロである宅建業者から一般消費者を保護するための制度です。したがって、売主が宅建業者で、買主が宅建業者でない場合にのみ適用されます。買主が宅建業者の場合は適用されません。

問3. 買主がクーリング・オフを行った場合の効果として、正しいものはどれか。

  • 1. 売主は、受領した手付金から違約金を差し引いて返還できる。
  • 2. 売主は、契約解除に伴う損害賠償を買主に請求できる。
  • 3. 売主は、受領した手付金その他の金銭を、全額、速やかに返還しなければならない。
  • 4. 買主は、契約解除の理由を売主に説明する義務がある。

正解は3

クーリング・オフは無条件解除なので、売主は損害賠償や違約金を請求することは一切できず、受領した金銭は全額返還しなければなりません。買主は理由を説明する必要もありません。

問4. クーリング・オフについて書面で告げられた日から、何日を経過するとクーリング・オフができなくなるか。

  • 1. 5日
  • 2. 8日
  • 3. 10日
  • 4. 14日

正解は2

クーリング・オフができる期間は、宅建業者から書面でその旨を告げられた日から起算して8日間です。この「8日」という日数は必ず覚えましょう。

問5. 買主の申出により、買主の勤務先で売買契約の申込みをした。この場合、クーリング・オフは適用されるか。

  • 1. 適用される。
  • 2. 適用されない。
  • 3. 買主が役員の場合に限り適用されない。
  • 4. 申込みの翌日であれば適用される。

正解は1

買主の自宅や勤務先は「事務所等以外」の場所に該当します。たとえ買主自らが場所を指定（申出）したとしても、そこが事務所等でなければクーリング・オフの対象となります。
