宅建業法 - 37条書面（契約書）

重要ポイント要約

  • 37条書面は、契約が成立した後、遅滞なく、契約の当事者双方に交付するもの。
  • 書面の交付は宅建業者の義務だが、説明の義務はない。
  • 書面には、取引に関わった宅地建物取引士の記名が必要（押印は不要）。
  • 記載事項には、必ず記載する「必要的記載事項」と、定めがある場合に記載する「任意的記載事項」がある。
  • 「代金の額、支払時期、支払方法」や「宅地の引渡しの時期」は37条書面の必要的記載事項。35条書面との違いを明確に。

詳細解説

1. 37条書面の基本

宅建業者は、宅地建物の売買・交換・貸借の契約が成立した場合、契約後のトラブルを防止するため、契約内容を明らかにした書面を作成し、当事者に交付する義務があります。これが「37条書面」です。一般的には契約書の役割を果たします。

  • 交付の時期：契約成立後、遅滞なく。
  • 交付の相手方：契約当事者の双方（売主と買主、貸主と借主）。
  • 宅建士の役割：書面への記名（説明義務はない）。

2. 記載事項（必要的記載事項と任意的記載事項）

37条書面の記載事項は、必ず記載しなければならない「必要的記載事項」と、当事者間で特約などの定めをした場合にのみ記載すればよい「任意的記載事項」に分かれます。

【必要的記載事項の例（売買・交換）】

  • 当事者の氏名・住所
  • 宅地または建物を特定するために必要な表示
  • 代金の額、その支払の時期及び方法
  • 宅地または建物の引渡しの時期
  • 移転登記の申請の時期

【任意的記載事項の例】

  • 代金等に関する金銭の貸借のあっせんが不成立のときの措置（ローン特約）
  • 契約不適合責任に関する定め
  • 損害賠償額の予定または違約金に関する定め

知識チェック

問1. 37条書面に関する記述のうち、宅地建物取引業法の規定によれば、誤っているものはどれか。

  • 1. 宅建業者は、契約当事者の双方に37条書面を交付しなければならない。
  • 2. 37条書面の交付にあたり、宅地建物取引士は相手方に内容を説明する義務がある。
  • 3. 代金の額や支払方法は、37条書面の必要的記載事項である。
  • 4. 宅建業者が自ら貸主となる建物の貸借契約では、37条書面の交付は不要である。

正解は2

37条書面について、宅地建物取引士は記名をする義務がありますが、重要事項説明（35条書面）と異なり、内容を説明する義務までは課されていません。よって2が誤りです。4は「自ら貸借」が宅建業に当たらないため正しい記述です。

問2. 次のうち、37条書面の必要的記載事項ではないものはどれか。（売買契約の場合）

  • 1. 当事者の氏名及び住所
  • 2. 代金の額
  • 3. ローンのあっせんが不成立のときの措置
  • 4. 物件の引渡しの時期

正解は3

ローン特約に関する定めは、当事者間でそのような定めをした場合にのみ記載すればよい「任意的記載事項」です。1,2,4は契約の根幹に関わる内容であり「必要的記載事項」です。

問3. 37条書面への宅地建物取引士の記名について、正しい記述はどれか。

  • 1. 専任の宅地建物取引士である必要はない。
  • 2. 当該取引に全く関与していない宅地建物取引士でも記名できる。
  • 3. 記名とともに、押印も必ず必要である。
  • 4. 記名は、法人の代表者が行わなければならない。

正解は1

37条書面に記名する宅地建物取引士は、その取引に関与した宅建士であればよく、専任である必要はありません。押印は法改正により不要となりました。

問4. 宅地建物取引業者が、宅地の売買の媒介を行う場合、37条書面の交付時期として最も適切なものはいつか。

  • 1. 契約の申込みを受けたとき
  • 2. 重要事項の説明を行ったとき
  • 3. 契約が成立した後、遅滞なく
  • 4. 物件の引渡しと同時に

正解は3

37条書面は、契約内容を確定させる書面であるため、契約が成立した「後」、遅滞なく交付しなければなりません。

問5. 宅地建物取引業者Aが、自ら売主として宅地建物取引業者でない買主Bと宅地の売買契約を締結した。この場合、Aは誰に37条書面を交付しなければならないか。

  • 1. Bにのみ交付すればよい。
  • 2. Bと、Bが指定する者に交付する。
  • 3. Bに交付する義務があるが、実務上はAも保管用に作成する。
  • 4. 宅建業者間の取引ではないので交付義務はない。

正解は3

37条書面は契約の当事者双方に交付する義務があります。この設問では、Aが自ら当事者（売主）なので、相手方である買主Bに交付する義務を負います。実務上は、売主Aと買主Bの両方に交付（同じものを2部作成し、署名・記名後に各1部保有）します。
