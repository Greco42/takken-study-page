① 宅建業法 (問1～問20)

問1. 宅地建物取引業の免許（以下、免許）に関する記述のうち、正しいものはどれか。

  • 1. A社が乙県のみに事務所を設置している場合でも、全国の物件を取り扱う場合は国土交通大臣の免許が必要である。
  • 2. 個人Bが破産手続開始の決定を受けたが、まだ復権を得ていない場合、Bは免許を受けることができない。
  • 3. C社の役員が刑法の傷害罪で罰金の刑に処せられた場合、刑の執行が終わってから5年を経過しないとC社は免許を受けられない。
  • 4. D社が甲県と乙県に事務所を設置する場合、甲県知事と乙県知事の両方の免許が必要である。

正解は2

1.免許は事務所の所在地で決まるため乙県知事免許でよい。3.宅建業法違反等による罰金刑は欠格事由だが、傷害罪での罰金刑は欠格事由ではない。4.複数の都道府県に事務所を置く場合は国土交通大臣免許が必要。

問2. 宅地建物取引士に関する記述のうち、誤っているものはどれか。

  • 1. 宅地建物取引士証の交付を受けていない宅地建物取引士は、重要事項の説明をすることはできない。
  • 2. 宅地建物取引士は、不正の手段により宅地建物取引士の登録を受けた場合、登録を消除されることがある。
  • 3. 宅地建物取引士は、事務禁止処分を受けた場合、速やかに宅地建物取引士証をその登録をしている都道府県知事に返納しなければならない。
  • 4. 宅地建物取引士は、常に自己の知識及び能力の向上に努めなければならない。

正解は3

事務禁止処分を受けた場合は、宅地建物取引士証を「提出」する義務があるが、「返納」するのは登録が消除された場合です。提出と返納の違いを覚えておきましょう。

問3. 営業保証金に関する記述のうち、正しいものはどれか。

  • 1. 本店と支店1か所を設置する場合、営業保証金は合計で1,500万円を供託しなければならない。
  • 2. 営業保証金は、金銭で供託しなければならず、有価証券で供託することはできない。
  • 3. 営業保証金を取り戻す場合、還付請求権者に対し、1年以上の期間を定めて申し出るべき旨を公告しなければならない。
  • 4. 営業保証金の還付により不足額が生じた場合、その通知を受けた日から1週間以内に補充しなければならない。

正解は1

1.本店1,000万円＋支店500万円＝1,500万円。2.国債証券や地方債証券など有価証券での供託も可能。3.公告期間は6ヶ月以上。4.補充期間は2週間以内。

問4. 宅地建物取引業保証協会に関する記述として、誤っているものはどれか。

  • 1. 宅建業者が保証協会に加入した場合、営業保証金の供託は免除される。
  • 2. 保証協会に加入しようとする宅建業者は、本店60万円、支店1か所につき30万円の弁済業務保証金分担金を納付する。
  • 3. 宅建業者との取引により損害を受けた者は、保証協会から弁済を受けるために、まずその宅建業者と訴訟を提起し勝訴判決を得る必要がある。
  • 4. 弁済業務保証金の還付があった場合、保証協会は当該社員に対し、還付充当金を納付するよう通知する。

正解は3

保証協会の認証があれば、必ずしも判決は必要ありません。

問5. 専任媒介契約を締結した宅地建物取引業者が、指定流通機構（レインズ）へ登録すべき事項に含まれないものはどれか。

  • 1. 当該宅地又は建物の所在、規模、形質
  • 2. 当該宅地又は建物を売買すべき価額
  • 3. 当該宅地又は建物に係る依頼者の住所、氏名
  • 4. 法令上の制限で重要なもの

正解は3

依頼者のプライバシー保護のため、依頼者の住所・氏名は登録事項に含まれません。

問6. 35条書面に記名する宅地建物取引士は、専任の宅地建物取引士でなければならない。

  • 1. 正しい
  • 2. 誤り

正解は2

重要事項の説明は宅地建物取引士が行いますが、その宅建士は専任である必要はありません。

問7. 37条書面の交付に関する記述のうち、正しいものはどれか。

  • 1. 契約の当事者が宅建業者である場合、37条書面の交付を省略できる。
  • 2. 37条書面には、宅地建物取引士が記名しなければならないが、内容を説明する義務はない。
  • 3. 建物の引渡しの時期は、35条書面と37条書面の両方に記載しなければならない。
  • 4. 37条書面は、契約が成立する前に交付しなければならない。

正解は2

1.相手が業者でも交付は省略不可。3.引渡時期は37条書面の必要的記載事項だが、35条書面では任意的記載事項。4.契約成立後、遅滞なく交付する。

問8. 宅建業者Aが自ら売主として宅建業者でないBに宅地を販売する。この場合、Aはどのような理由があっても、Bに損害賠償額の予定と違約金の合計額として、代金の20%を超える額を請求することはできない。

  • 1. 正しい
  • 2. 誤り

正解は1

自ら売主制限の一つとして、損害賠償額の予定等の制限があります。代金の20%を超える部分は無効となります。

問9. クーリング・オフに関する記述のうち、正しいものはどれか。

  • 1. 買主は、クーリング・オフを行う旨の書面が売主に到達した時点から、契約を解除できる。
  • 2. 売主は、クーリング・オフによる解除を受けた場合、受領した手付金その他の金銭を速やかに全額返還しなければならない。
  • 3. 買主が、自らの申出により自宅で契約した場合、クーリング・オフはできない。
  • 4. 買主が物件の引渡しを受ければ、代金全額を支払っていなくてもクーリング・オフはできない。

正解は2

1.効力は書面を発信した時に生じる。3.買主の申出があっても自宅は対象外ではない。4.引渡し「かつ」代金全額支払で初めてできなくなる。

問10. 宅地建物取引業者が、代金400万円（税抜）の宅地の売買を媒介した場合、依頼者の一方から受け取れる報酬の上限額（消費税抜）はいくらか。

  • 1. 14万円
  • 2. 18万円
  • 3. 20万円
  • 4. 22万円

正解は2

取引額が200万円超400万円以下の速算式「取引額 × 4% + 2万円」を使います。400万円 × 4% + 2万円 = 16万円 + 2万円 = 18万円となります。

  • 2. 誤り

正解は1

これは専任媒介契約等を締結した業者の義務として定められています。

問12. 宅建業者Aが、自ら売主として宅建業者でない買主Bに建築工事完了前の物件を販売する場合、Aが手付金等の保全措置を講じずに受領できる手付金の上限は、代金の5%かつ1,000万円以下である。

問15. 宅建業者は、一団の宅地の分譲をする案内所において、契約の申込みを受けない場合でも、専任の宅地建物取引士を1名以上置かなければならない。

問18. 宅建業者A社が、免許の更新申請を怠り、有効期間が満了した。A社は、更新申請を行っていなかった期間中に締結した契約については、取引を結了する目的の範囲内において、なお宅建業者とみなされる。

② 権利関係 (問21～問34)

問21. AがBの詐欺により、自己所有の土地をBに売却した。その後、Bがその事情を知らない（善意の）Cに土地を転売した場合、AはCに対して土地の返還を請求できるか。

  • 1. Cが善意であれば、返還を請求できない。
  • 2. Cが善意無過失であれば、返還を請求できない。
  • 3. Cが善意であっても、返還を請求できる。
  • 4. Cが悪意の場合のみ、返還を請求できる。

正解は2

詐欺による意思表示の取消しは、善意でかつ過失がない第三者に対抗できません。強迫の場合と異なり、第三者に無過失まで要求されるのがポイントです。

問22. Aの代理人と称する無権代理人Bが、Aの土地をCに売却した。Aがこの契約を追認しない場合、CはBに対してどのような請求ができるか。

  • 1. 常に損害賠償を請求できる。
  • 2. 契約の履行または損害賠償を請求できる。
  • 3. 契約の履行のみを請求できる。
  • 4. Cは何も請求できない。

正解は2

無権代理人の責任として、相手方Cは、Bに対して契約を履行するよう求めるか、または損害賠償を請求するかを選択できます。ただし、Cが悪意であった場合などは請求できません。

問23. A所有の土地をBが占有している。Bの占有が、所有の意思をもって、平穏かつ公然に、善意無過失で開始された場合、Bが所有権を時効取得するのに必要な期間はどれか。

  • 1. 10年
  • 2. 15年
  • 3. 20年
  • 4. 30年

正解は1

善意・無過失で占有を開始した場合の取得時効の期間は10年です（短期取得時効）。

問24. AがBに土地を売却し、Bが登記を備えていない間に、Aが同じ土地をCに売却し、Cが先に登記を備えた。この場合、BはCに対して所有権を主張できるか。

  • 1. Bが先に契約したので、主張できる。
  • 2. Cが先に登記を備えたので、主張できない。
  • 3. Cが悪意（二重譲渡の事実を知っていた）であれば、主張できる。
  • 4. 契約の有効性は裁判で決まる。

正解は2

不動産の二重譲渡では、契約の先後ではなく、先に登記を備えた方が所有権を主張できます。Cが悪意であっても、登記があればCが勝ちます。

問25. AがBから借り入れた債務を担保するため、A所有の建物に抵当権を設定した。Aが債務を返済しない場合、Bはどうすることができるか。

  • 1. 抵当権を実行し、建物を競売にかけて、その売却代金から優先的に弁済を受けることができる。
  • 2. 直ちに建物の所有権を取得することができる。
  • 3. 建物を占有し、賃貸して収益を上げることができる。
  • 4. Aに代わって建物を売却することができる。

正解は1

抵当権の主な効力は、目的物を競売にかけ、その代金から他の債権者に優先して弁済を受ける「優先弁済権」です。

問26. 請負人Aが、注文者Bから請け負って建築した新築住宅に、構造耐力上の瑕疵（契約不適合）があった。この場合、BがAに対して請求できないものはどれか。

  • 1. 追完請求（修補の請求）
  • 2. 代金減額請求
  • 3. 損害賠償請求
  • 4. 新築住宅の取替え請求

正解は4

契約不適合責任として、買主（注文者）は追完請求、代金減額請求、損害賠償請求、契約解除ができます。しかし、一品製作物である不動産において、代替物の引渡し（取替え請求）は認められていません。

問27. 連帯債務者の一人が債権者から履行の請求を受けた場合、他の連帯債務者の債務の時効はどうなるか。

  • 1. 他の連帯債務者の債務の時効も更新される。
  • 2. 他の連帯債務者の債務の時効には影響しない。
  • 3. 他の連帯債務者の債務の時効は完成猶予となる。
  • 4. 請求された債務者の負担部分についてのみ更新される。

正解は1

連帯債務では、一人に対する履行の請求は、他の債務者に対しても効力を生じます（絶対効）。したがって、時効も更新されます。

問28. Aが所有する建物をBに賃貸した。BがAの承諾を得ずに、この建物をCに無断で転貸した場合、Aはどうすることができるか。

  • 1. Bに損害賠償を請求できるのみである。
  • 2. Cに対して、建物の明渡しを請求できる。
  • 3. AB間の信頼関係を破壊する特段の事情があれば、賃貸借契約を解除できる。
  • 4. Aは何もできない。

正解は3

賃借権の無断譲渡・転貸は、賃貸人に対する裏切り行為（背信行為）にあたるため、賃貸人は契約を解除できます。ただし、判例上、信頼関係を破壊するに至らない特段の事情がある場合は解除が認められないことがあります。

問29. 建物の区分所有法に関する記述のうち、誤っているものはどれか。

  • 1. 集会（総会）は、少なくとも毎年1回、定期的に招集しなければならない。
  • 2. 共用部分の変更（その形状又は効用の著しい変更を伴わないものを除く）には、区分所有者及び議決権の各4分の3以上の多数による集会の決議が必要である。
  • 3. 管理者は、規約に別段の定めがない限り、共用部分の保存行為を行うために、集会の決議を経る必要がある。
  • 4. 建物の価格の2分の1を超える部分が滅失した場合、各区分所有者は、滅失した共用部分を復旧する旨の決議をすることができる。

    • 正解は3

    共用部分の保存行為（現状を維持するための行為）は、各共有者が単独で行うことができ、管理者が行う場合も集会の決議は不要です。

    問30. Aが死亡し、相続人が配偶者Bと弟Cのみである場合、Bの法定相続分はどれだけか。

    • 1. 1/2
    • 2. 2/3
    • 3. 3/4
    • 4. 全て

    正解は3

    相続人が配偶者と兄弟姉妹（第3順位）の場合、法定相続分は配偶者3/4、兄弟姉妹1/4となります。

    問31. 普通借地権の存続期間が満了した場合、借地権者が契約の更新を請求した。建物が存在する場合、借地権設定者はどのような場合に更新を拒絶できるか。

    • 1. 常に拒絶できる。
    • 2. 地代の増額に応じない場合。
    • 3. 遅滞なく、自己使用の必要性などの正当事由をもって、異議を述べた場合。
    • 4. 理由なく拒絶できる。

    正解は3

    普通借地権では、借地人を保護するため、地主からの更新拒絶には、立ち退き料の提供なども考慮した「正当事由」が必要です。

    問32. 定期借家契約を締結する際、貸主が借主に対して「この契約は更新がなく、期間の満了により終了する」旨を記載した書面を交付して説明することを怠った場合、その契約はどうなるか。

    • 1. 契約は無効となる。
    • 2. 定期借家契約として有効に成立する。
    • 3. 普通借家契約となり、更新の可能性がある契約となる。
    • 4. 借主はいつでも契約を解除できる。

    正解は3

    定期借家契約の事前説明は極めて重要な要件であり、これを怠ると、その契約は更新のある「普通借家契約」とみなされてしまいます。

    問33. 不動産の登記に関する記述として、正しいものはどれか。

    • 1. 表示に関する登記には、権利に関する登記のような対抗力はない。
    • 2. 仮登記には、順位を保全する効力はない。
    • 3. 建物を新築した場合、1ヶ月以内に表題登記を申請しなければならないが、罰則はない。
    • 4. 不動産の登記は、オンラインで申請することはできない。

    正解は1

    表示に関する登記（土地の地目や面積、建物の種類や床面積など）は、不動産の物理的現況を示すものであり、権利の優劣を決める対抗力はありません。

    問34. 被保佐人が、保佐人の同意を得ずに行った行為でも、有効な法律行為は次のうちどれか。

    • 1. 不動産を売却すること
    • 2. 金銭を借り入れること
    • 3. 10万円のテレビを購入すること
    • 4. 他人の保証人になること

    正解は3

    被保佐人が保佐人の同意を必要とするのは、借金、保証、不動産の売買など、民法13条1項で定められた「特に重要な財産上の行為」です。テレビの購入のような、これらに該当しない行為は、保佐人の同意なく単独で有効に行うことができます。

    ③ 法令上の制限 (問35～問42)

    問35. 市街化区域内において、2,000㎡の土地でゴルフ場を建設するための開発行為を行う場合、開発許可は必要か。

    • 1. 必要である。
    • 2. 不要である。

    正解は2

    第二種特定工作物であるゴルフコースの建設は、市街化区域内では規模にかかわらず開発許可が不要です。ただし、1ha以上の場合は必要となります。

    問36. 建築基準法上の道路に関する記述として、誤っているものはどれか。

    • 1. 建築物の敷地は、原則として幅員4m以上の道路に2m以上接しなければならない。
    • 2. 幅員が4m未満であっても、特定行政庁が指定した道（2項道路）に接していれば、セットバックは不要である。
    • 3. 自動車専用道路は、建築基準法上の道路ではない。
    • 4. 都市計画により新たに築造される道路も、建築基準法上の道路である。

    正解は2

    2項道路に接する場合は、原則として道路中心線から2mの線を道路境界線とみなすため、セットバックが必要です。

    問37. 第一種低層住居専用地域内に建築することができないものは、次のうちどれか。

    • 1. 診療所
    • 2. コンビニエンスストア
    • 3. 公衆浴場
    • 4. 小学校

    正解は2

    コンビニエンスストアのような物品販売業を営む店舗は、第一種低層住居専用地域では建築できません。

    問38. 防火地域内にある、階数が2で延べ面積が150㎡の建築物は、原則としてどのような構造にしなければならないか。

    • 1. 準耐火建築物
    • 2. 耐火建築物
    • 3. 防火構造
    • 4. 特に制限はない。

    正解は2

    防火地域内では、階数が3以上、または延べ面積が100㎡を超える建築物は耐火建築物としなければなりません。この建物は延べ面積が150㎡なので、耐火建築物にする必要があります。

    問39. 国土利用計画法の事後届出について、市街化区域内の2,500㎡の土地を売買した場合、届出は必要か。

    • 1. 必要である。
    • 2. 不要である。

    正解は1

    事後届出が必要な面積は、市街化区域では2,000㎡以上です。2,500㎡の取引なので届出が必要です。

    問40. 農業を営むAが、自己所有の農地（市街化調整区域内）に、農業用倉庫を建てるために転用する場合、農地法4条の許可は必要か。

    • 1. 必要である。
    • 2. 不要である。

    正解は2

    自己の農地を、農業用施設（畜舎、温室、農作業小屋など）に転用する場合、農地法4条の許可は不要です。

    問41. 宅地造成等規制法に関する記述として、正しいものはどれか。

    • 1. 宅地造成工事規制区域内で行う、高さ1.5mの擁壁の設置工事は、許可が不要である。
    • 2. 宅地造成工事規制区域内で行う、高さ2.5mの切土であって、切土面積が400㎡の工事は、許可が必要である。
    • 3. 宅地造成工事規制区域の指定は、国土交通大臣が行う。
    • 4. 造成主は、工事完了後、2週間以内に検査を受けなければならない。

    正解は2

    許可が必要な工事は、①高さ2m超の切土、②高さ1m超の盛土、③切土と盛土を同時に行い、盛土が1m以下でも合計の高さが2m超、④切土・盛土に関わらず面積500㎡超。2は高さ2.5mの切土なので許可が必要です。

    問42. 土地区画整理組合が施行する土地区画整理事業において、換地処分は、換地計画に係る区域の全部について工事が完了した後に、遅滞なく行わなければならない。

    • 1. 正しい
    • 2. 誤り

    正解は1

    換地処分は、原則として工事完了後に行われます。この換地処分の公告のあった日の翌日から、従前の宅地の権利は換地処分後の土地に移行します。

    ④ 税・その他 (問43～問50)

    問43. 相続によって不動産を取得した場合、不動産取得税は課税されるか。

    • 1. 課税される。
    • 2. 課税されない。

    正解は2

    相続による取得は不動産取得税の非課税事由です。

    問44. 固定資産税の納税義務者は、原則としてその年のいつ時点の所有者か。

    • 1. 1月1日
    • 2. 4月1日
    • 3. 登記をした日
    • 4. 契約をした日

    正解は1

    固定資産税は、毎年1月1日（賦課期日）時点の所有者に対して課税されます。

    問45. 印紙税が課税されない文書は、次のうちどれか。

    • 1. 土地の売買契約書
    • 2. 建物の賃貸借契約書
    • 3. 建築工事請負契約書
    • 4. 1,000万円の金銭消費貸借契約書

    正解は2

    不動産の賃貸借契約書は、課税文書に該当しないため非課税です。

    問46. 登録免許税の住宅用家屋の軽減措置を受けるための床面積の要件として、正しいものはどれか。

    • 1. 30㎡以上
    • 2. 50㎡以上
    • 3. 50㎡以上280㎡以下
    • 4. 特に要件はない。

    正解は2

    住宅用家屋の軽減措置を受けるには、その家屋の登記上の床面積が50㎡以上であることが要件の一つです。

    問47. 個人が、居住用財産を譲渡した場合の3,000万円の特別控除の適用を受けるためには、譲渡した年の1月1日において、所有期間が10年を超えている必要がある。

    • 1. 正しい
    • 2. 誤り

    正解は2

    3,000万円の特別控除には、所有期間の長短は問われません。所有期間10年超の要件があるのは、軽減税率の特例です。

    問48. 地価公示は、誰が、いつ時点の価格を公示するものか。

    • 1. 都道府県知事が、毎年7月1日時点
    • 2. 土地鑑定委員会が、毎年1月1日時点
    • 3. 市町村長が、3年に1度、1月1日時点
    • 4. 国土交通大臣が、毎年4月1日時点

    正解は2

    地価公示は、国土交通省の土地鑑定委員会が、毎年1月1日時点の価格を判定し公示します。

    問49. 不動産鑑定評価における鑑定評価の手法でないものはどれか。

  • 1. 原価法
  • 2. 取引事例比較法
  • 3. 収益還元法
  • 4. 路線価法

    • 正解は4

    鑑定評価の基本的な手法は、原価法、取引事例比較法、収益還元法の3つです。路線価法は、相続税や贈与税の算定に用いられる土地の評価方法の一つです。

    問50. 住宅金融支援機構の【フラット３５】は、新築住宅の建設・購入資金だけでなく、中古住宅の購入資金にも利用することができる。

    • 1. 正しい
    • 2. 誤り

    正解は1

    【フラット３５】は、新築住宅だけでなく、一定の技術基準を満たす中古住宅の購入資金や、リフォーム資金にも利用できます。
