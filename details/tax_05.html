税・その他 - 地価公示法

重要ポイント要約

  • 土地鑑定委員会が、都市計画区域内の標準地を選定する。
  • 毎年1回、その標準地の1月1日時点における単位面積あたりの正常な価格を判定し、公示する。これを公示価格という。
  • 公示価格は、一般の土地取引の際の指標となる。
  • 公共事業の用地買収価格は、原則として公示価格を基準として算定される。
  • 宅建業者は、近傍の標準地の公示価格を、取引の相手方に教えるよう努めなければならない（努力義務）。

詳細解説

1. 地価公示制度の目的

地価公示法は、都市計画区域内等の標準的な土地（標準地）について、毎年1月1日時点の正常な価格（公示価格）を一般に公表（公示）する制度です。この公示価格を指標とすることで、適正な地価の形成に役立てることを目的としています。

2. 地価公示の手続き

  1. 標準地の選定：国土交通省に設置された土地鑑定委員会が、土地の利用状況が標準的と認められる「標準地」を選定します。
  2. 鑑定評価：土地鑑定委員会が、2人以上の不動産鑑定士に鑑定評価を依頼します。
  3. 価格の判定と公示：土地鑑定委員会が、鑑定評価を審査・調整し、毎年1月1日時点における単位面積（1㎡）あたりの正常な価格を判定し、公示します。

3. 公示価格の効力・役割

公示価格には、法的な拘束力はありませんが、以下の重要な役割を果たします。

  • 一般の土地取引の指標：土地取引を行う者は、公示価格を指標として取引を行うよう努めなければならない、とされています。
  • 公共事業用地の取得価格の算定基準：国や地方公共団体などが公共事業のために土地を取得する場合、公示価格を基準として価格を算定しなければなりません。
  • 不動産鑑定評価の基準：不動産鑑定士が土地の鑑定評価を行う場合、公示価格を基準としなければなりません。

知識チェック

問1. 地価公示の価格判定の基準日はいつか。

  • 1. 毎年1月1日
  • 2. 毎年4月1日
  • 3. 毎年7月1日
  • 4. 3年に1回、1月1日

正解は1

地価公示の価格判定の基準日は、毎年1月1日です。固定資産税の賦課期日と同じ日なので、関連付けて覚えましょう。

問2. 地価公示を行う機関はどこか。

  • 1. 各都道府県
  • 2. 各市町村
  • 3. 国土交通大臣
  • 4. 土地鑑定委員会

正解は4

地価公示は、国土交通省に置かれる土地鑑定委員会が行います。都道府県知事が行うのは「地価調査」（基準地価格）であり、地価公示とは別の制度です。

問3. 公示価格は、どのような土地取引の価格の指標となるよう定められているか。

  • 1. 公共事業用地の取引のみ
  • 2. 宅地建物取引業者間の取引のみ
  • 3. 一般の土地の取引
  • 4. 相続税の算定基準となる取引

正解は3

公示価格は、公共事業の用地取得価格の「基準」となるほか、広く一般の土地取引の「指標」となることが期待されています。

問4. 標準地の鑑定評価を行うのは誰か。

  • 1. 土地鑑定委員会の委員
  • 2. 2人以上の不動産鑑定士
  • 3. 1人以上の宅地建物取引士
  • 4. 当該標準地が所在する市町村の長

正解は2

土地鑑定委員会は、標準地の鑑定評価を2人以上の不動産鑑定士に依頼し、その結果を審査・調整して公示価格を決定します。

問5. 宅地建物取引業者は、宅地の売買にあたり、近傍の標準地の公示価格を取引の相手方に説明する義務があるか。

  • 1. 必ず説明しなければならない義務がある。
  • 2. 説明するよう努めなければならない（努力義務）。
  • 3. 相手方から請求があった場合のみ説明する義務がある。
  • 4. 説明する義務は全くない。

正解は2

宅地建物取引業法では、宅地建物の売買等にあたり、公示価格を指標として取引を行うよう、当事者に対して助言するよう「努めなければならない」とされています。これは努力義務であり、絶対的な義務ではありません。
