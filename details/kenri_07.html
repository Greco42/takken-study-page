権利関係 - 相続

重要ポイント要約

  • 法定相続人：配偶者は常に相続人。子・直系尊属（親など）・兄弟姉妹の順で相続人になる。
  • 法定相続分：法律で定められた相続分の割合。配偶者と子の場合は、それぞれ1/2ずつ。
  • 遺言：被相続人（亡くなった人）の最終の意思表示。遺言があれば、原則として法定相続より優先される。
  • 遺留分：兄弟姉妹を除く法定相続人に保障された、最低限の遺産の取り分。遺言によっても奪うことはできない。
  • 遺産分割：相続人が複数いる場合、遺産の分け方を決める話し合い。分割協議がまとまるまで、不動産は相続人全員の共有状態となる。

詳細解説

1. 法定相続人とその順位

誰が相続人になるかは法律で定められています。

  • 配偶者：常に相続人となります。
  • 血族相続人：以下の順位で、上位の人がいる場合は下位の人は相続人になれません。
    • 第1順位：子（子が先に死亡している場合は孫などの代襲相続人）
    • 第2順位：直系尊属（父母、祖父母など）
    • 第3順位：兄弟姉妹（兄弟姉妹が先に死亡している場合はその子である甥・姪が代襲相続人）

2. 法定相続分

遺言がない場合の、各相続人の取り分の割合です。

  • 配偶者と子：配偶者 1/2、子 1/2（子が複数いれば均等割り）
  • 配偶者と直系尊属：配偶者 2/3、直系尊属 1/3
  • 配偶者と兄弟姉妹：配偶者 3/4、兄弟姉妹 1/4

3. 遺言と遺留分

遺言があれば、法定相続の規定にかかわらず、原則として遺言の内容通りに遺産が分けられます。しかし、相続人の生活保障などのため、遺留分という制度があります。

遺留分は、兄弟姉妹を除く法定相続人に認められた最低限の遺産の取り分です。遺言によって自分の遺留分が侵害された相続人は、遺産を多く受け取った人に対して、侵害された額に相当する金銭の支払いを請求できます（遺留分侵害額請求）。

4. 遺産分割

相続人が複数いる場合、相続財産は一時的に相続人全員の共有状態となります。この共有状態を解消し、各相続人の具体的な取得財産を決める手続きが遺産分割です。遺産分割は、相続人全員の協議によって行います。

【宅建試験での重要ポイント】

共有状態の不動産を売却したり、大規模なリフォームをしたりするには、共有者（共同相続人）全員の同意が必要です。一人でも反対すれば、売却等はできません。

知識チェック

問1. 被相続人Aには、配偶者Bと子C、Dがいる。Aが遺言なく死亡した場合の法定相続分として正しいものはどれか。

  • 1. Bが2/3、CとDがそれぞれ1/6
  • 2. B、C、Dがそれぞれ1/3ずつ
  • 3. Bが1/2、CとDがそれぞれ1/4
  • 4. Bが3/4、CとDがそれぞれ1/8

正解は3

配偶者と子（第1順位）が相続人の場合、法定相続分は配偶者1/2、子1/2です。子が複数いる場合は、子の取り分である1/2を均等に分けます。したがって、CとDはそれぞれ1/2 × 1/2 = 1/4となります。

問2. 遺留分を有する相続人は、次のうち誰か。

  • 1. 被相続人の子
  • 2. 被相続人の兄弟姉妹
  • 3. 被相続人の甥
  • 4. 被相続人のいとこ

正解は1

遺留分が認められているのは、兄弟姉妹を除く法定相続人です。具体的には、配偶者、子（およびその代襲相続人）、直系尊属です。兄弟姉妹とその代襲相続人（甥・姪）には遺留分はありません。

問3. A、B、Cが不動産を共同相続した。この不動産を売却するには、どのような合意が必要か。

  • 1. A、B、Cの持分の価格に従い、その過半数の同意
  • 2. A、B、Cのうち、2名の同意
  • 3. A、B、Cの全員の同意
  • 4. 代表者Aの単独の判断

正解は3

共同相続により共有となった不動産全体を売却するなどの処分行為は、共有者全員の同意がなければ行うことができません。

問4. 被相続人に配偶者と弟がいる場合の法定相続人は誰か。

  • 1. 配偶者のみ
  • 2. 弟のみ
  • 3. 配偶者と弟
  • 4. 該当者なし

正解は3

配偶者は常に相続人です。子（第1順位）と直系尊属（第2順位）がいない場合、第3順位である兄弟姉妹が相続人となります。したがって、配偶者と弟が相続人となります。

問5. 自筆証書遺言に関する記述として、誤っているものはどれか。

  • 1. 遺言者が、その全文、日付及び氏名を自書しなければならない。
  • 2. 遺言者が、遺言書に押印しなければならない。
  • 3. 証人が2人以上立ち会わなければならない。
  • 4. 家庭裁判所による検認の手続きが必要である。（法務局保管制度利用時を除く）

正解は3

証人2人以上の立会いが必要なのは、公正証書遺言の要件です。自筆証書遺言は、証人不要で作成できます。
