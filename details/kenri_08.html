権利関係 - 代理

重要ポイント要約

  • 代理とは、本人の代わりに行動し、その法律効果を本人に直接発生させる制度。
  • 代理人が契約する際、「これは本人のための契約です」と示すこと（顕名）が必要。
  • 代理人が本人をだましても、相手方がその事実を知らなければ、本人は契約を取り消せない。
  • 無権代理：代理権がないのに代理人として行為すること。原則として本人に効果は及ばず無効。
  • 本人は無権代理行為を追認して有効にできる。相手方は本人に追認するかどうかを催告できる。

詳細解説

1. 代理の仕組み（三面関係）

代理は、①本人、②代理人、③相手方の三者間で成立します。代理人が、本人の代理人であることを相手方に示し（顕名）、代理権の範囲内で相手方と契約などをすると、その効果（権利や義務）は、代理人ではなく本人と相手方の間に直接発生します。

例：売主A（本人）がB（代理人）に土地売却の代理権を与え、Bが買主C（相手方）と売買契約を結ぶと、AとCの間で売買契約が成立します。

2. 代理行為の要件と効果

  • 顕名（けんめい）：代理人は、相手方に対して「本人のためにする」という意思表示をしなければなりません。顕名をしない場合、原則として代理人自身の行為とみなされます。
  • 代理権の範囲内：代理人は、与えられた代理権の範囲内でしか行動できません。

これらの要件を満たした代理行為の効果は、すべて直接本人に帰属します。たとえ代理人が本人をだますつもり（背任的意図）で代理行為をしても、相手方がその意図を知っていたり、知ることができた場合でない限り、契約は有効に本人に帰属します。

3. 無権代理

代理権がないにもかかわらず、代理人と称して行われた行為を「無権代理」といいます。無権代理行為は、本人に効果が帰属せず、原則として無効です。

しかし、本人の意思や相手方の保護のために、以下の制度が設けられています。

  • 本人の追認権：本人は、無権代理行為を後から認める（追認する）ことで、契約を有効なものとして確定させることができます。
  • 相手方の催告権：相手方は、本人に対して、相当の期間内に追認するかどうかを確答するよう催告できます。期間内に本人が返事をしない場合、追認を拒絶したものとみなされます。
  • 相手方の取消権：相手方は、無権代理人が代理権を有しないことを知らなかった（善意である）場合に限り、本人が追認する前に契約を取り消すことができます。

4. 表見代理

無権代理であるにもかかわらず、あたかも有効な代理権があるかのような外観（見た目）が存在し、その外観を信じた相手方を保護するため、本人に契約の効果を帰属させる制度です。例えば、本人が「Bを代理人にした」とCに伝えていたが、実際には代理権を与えていなかった場合などが該当します。

知識チェック

問1. Aの代理人Bが、Cとの間でA所有の甲土地の売買契約を締結した。この契約の効果がAに帰属するために、BがCに対して示すべきことは何か。

  • 1. B自身の資力
  • 2. Aのためにする意思（顕名）
  • 3. Aの印鑑証明書
  • 4. 契約を締結する理由

正解は2

代理行為の効果が本人に帰属するためには、代理人が相手方に対して、その行為が本人のためになされるものであることを示す「顕名」を行う必要があります。

問2. 代理権を有しないDが、Aの代理人と称してA所有の土地をEに売却した（無権代理）。この場合、Aが取りうる行動として正しいものはどれか。

  • 1. この売買契約を追認して、有効な契約とすることができる。
  • 2. Dに対して損害賠償を請求することしかできない。
  • 3. この契約を一方的に取り消すことができる。
  • 4. Eに対して、直ちに土地の明渡しを請求することはできない。

正解は1

無権代理行為に対し、本人は追認権を持っています。追認すれば、契約ははじめにさかのぼって有効であったものとして扱われます。なお、追認を拒絶することも可能です。

問3. 無権代理行為の相手方Eは、Dに代理権がないことを知らなかった（善意）。この場合、Eができることは何か。

  • 1. Aが追認した後でも、契約を取り消すことができる。
  • 2. Dに対して、Aに追認するよう説得を求めることができる。
  • 3. Aが追認する前に限り、契約を取り消すことができる。
  • 4. 契約を取り消すことはできず、Aの追認を待つしかない。

正解は3

無権代理の相手方は、代理人に代理権がないことを知らなかった（善意の）場合、本人が追認をする前に、契約を取り消すことができます。

問4. 無権代理行為の相手方Eが、Aに対して相当の期間を定めて追認するかどうかの催告をしたが、期間内にAから返答がなかった。この場合、法律上どう扱われるか。

  • 1. 追認したものとみなされる。
  • 2. 追認を拒絶したものとみなされる。
  • 3. 契約は自動的に無効となる。
  • 4. 催告の期間が延長される。

正解は2

相手方からの催告に対し、本人が期間内に確答を発しない場合、その本人は追認を拒絶したものとみなされます。契約が有効になることはありません。

問5. Aの代理人Bが、Cをだまして不当に安い価格でCの土地を買い受ける契約を締結した。Aがその事実を知らなかった場合、Cはこの契約を取り消せるか。

  • 1. Aが善意なので、Cは取り消せない。
  • 2. 代理人が行った詐欺なので、Cは取り消せる。
  • 3. BがAに事実を報告するまで、Cは取り消せない。
  • 4. AとCの協議による。

正解は2

代理人が相手方に対して詐欺を行った場合、その事実は本人と同一視されます。したがって、たとえ本人であるAがその詐欺の事実を知らなかったとしても、相手方Cは契約を取り消すことができます。
